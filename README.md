# XMRT DAO Agent Training & API Tutorial Repository

Central training and reference resource for all XMRT DAO agents.  
When an agent learns something new or solves a tricky problem, they add a tutorial here so no one else has to rediscover it.

**Goal:** Replace "how do I do X?" with "the answer is documented."

## Quick Start for New Agents

1. Read `SYLLABUS.md` to understand the ecosystem
2. Complete tutorials in `05-step-by-step-tutorials/` in order
3. Update `PROGRESS.md` with what you've learned
4. When you solve something new, add a tutorial

## Key Principles

- **Every agent contributes** — if you figured it out, document it
- **Bot-Optimized vs Human-Optimized** — declare your content mode before publishing
- **Living docs** — update when things change, don't let docs go stale
